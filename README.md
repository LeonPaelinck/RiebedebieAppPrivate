# RiebedebieApp

## Vereisten

- Node.js installed
  - https://nodejs.org/en/
- Angular CLI installed
  - `npm install -g @angular/cli`
- DotNet 5.0 Installed
- SQL Database installed on localhost 

## Development Opstarten

````
git clone deze repo

````

### Server

```
cd Server/RecipeApi
dotnet watch run
```


### Client

```
cd Client
npm install
ng build
npm start
```

---

## Productie opstarten

### Client

```
cd Client
npm install
ng build --prod
```

### Server

```
cd Server/RecipeApi
dotnet run
```
